# User Finder App

A JavaScript application to search GitHub users and view their profiles and top repositories.

## Features
- Search GitHub users by username
- View user profile details (name, avatar, bio)
- Show top repositories
- Dark mode toggle
- Debounced search for better performance

## Technologies Used
- HTML
- CSS
- JavaScript (ES6)
- Fetch API
